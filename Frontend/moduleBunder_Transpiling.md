# 모듈 번들러와 트랜스파일러에 대해서

> 모듈 번들러와 트랜스파일러는 현대 웹 개발에서 중요한 도구이다. 이들은 각각 웹 애플리케이션의 소스코드를 최적화 하고, 다양한 브라우저 환경에서 호환되도록 하는 역할을 한다.

## 모듈 번들러(Module Bundler)

- 모듈 번들러는 여러 개의 자바스크립트 파일(모듈)을 하나 또는 여러개의 파일로 합치는 도구이다.
- 웹 애플리케이션은 일반적으로 많은 모듈로 구성되어 있으며, 모듈 번들러는 이러한 모듈들을 브라우저가 이해할 수 있는 단일 파일로 결합한다.
- 모듈 번들러를 사용하면 네트워크 요청 횟수를 줄이고, 파일 크기를 최적화 하며, 모듈 종속성을 관리할 수 있다.
- 대표적인 모듈 번들러로는 Webpack, Parcel, Rollup등이 있다.

## 트랜스파일러(Transpiler)

- 트랜스파일러는 소스 코드를 같은 수준의 다른 언어로 변환하는 도구이다.
- 주로 최신 자바스크립트 문법을 구형 브라우저에서도 호환되는 형태로 변환하는 데 사용한다.
- 트랜스파일러는 JSX 같은 비표준 코드를 자바스크립트 코드로 변환하는 데에도 사용된다.
- 대표적인 트랜스파일러로는 Babel이 있다.
