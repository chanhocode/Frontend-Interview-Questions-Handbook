# ES6(ES2015)이후로 추가된 기능

- let과 const
- 클래스 구문
- 화살표 함수
- 템플릿 리터널
  - 문자열 삽입을 위한 템플릿 리터널(```)을 사용하여 표현식을 문자열에 쉽게 삽입할 수 있다.
- 기본 매개변수
  - 함수 매개변수에 기본값을 설정할 수 있다.
- 비구조화할당(Destructuring)
  - 객체 또는 배열의 속성이나 요소를 변수에 쉽게 할당할 수 있는 문법을 제공한다.
- Spread & Rest
- Promise 와 async/await
- 모듈
- Map, Set, WeakMap, WeakSet
