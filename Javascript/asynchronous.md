# 비동기 처리의 개념과 중요성
> 자바스크립트에서 비동기 처리란, 특정 코드의 실행이 완료될 때까지 기다리지 않고 다음코드를 실행하는 것을 의미한다. 이는 자바스크립트가 단일 스레드로 작동하기에 중요한 특징이다. 비동기 처리는 사용자 인터페이스 블록킹을 방지하고 효율적인 웹 애플리케이션 처리를 위해 필수적이다. 예들 들어, 서버로 부터 데이터를 받아오는 동안에도 사용자가 웹페이지를 자유롭게 탐색할 수 있게 해준다.

## 자바스크립트에서 비동기 처리가 왜 중요한가?
> 자바스크립트는 단일 스레드 언어로, 한 번에 하나의 작업만 처리할 수 있다. 비동기 처리를 통해 네트워크 요청, 파일 I/O등의 시간이 많이 소요되는 작업을 백그라운드에서 수행하고, 그 결과를 나중에 받을 수 있다. 이는 웹페이지가 더 빠르고 반응적으로 동작하도록 해주며, 사용자 경험을 향상시킵니다.

## 동기와 비동기
> 동기는 순차적으로 태스크를 수행한다. 요청 과 응답이 한셋트로 해당 셋트를 전부 수행해야 다음 동작을 한다. 이 처럼 순차적으로 실행되므로, 어떤 작업이 수행중에는 뒤의 작업들은 대기하게 된다. 비동기는 병렬적으로 태스크를 수행하며 동기 방식과 달리 완료 순서가 보장되지 않는다.
