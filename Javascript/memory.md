# 자바스크립트의 메모리 관리

> 자바스크립트는 `가비지 콜렉션(Garbage collection)` 메커니즘을 사용하여 메모리 관리를 자동으로 수행합니다. 이는 개발자가 수동으로 메모리를 할당하고 해제할 필요가 없게 해줍니다.

## 가비지 콜렉션

> 가비지 콜렉션은 프로그램에서 `더 이상 사용되지 않는 메모리`, 즉 "가비지"를 자동으로 검출하고 회수하는 프로세스 입니다. 자바스크립트 엔진은 도달 가능성 기준을 사용하여 메모리를 관리합니다. 도달 가능한 값은 어떤 방식으로든 접근하거나 사용할 수 있는 값으로, 이러한 값들은 메모리에 계속 유지 됩니다.
