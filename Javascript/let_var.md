# let 과 var의 차이

1. Scope

   > let은 `Block Scope`를 가지고, var은 `Function Scope`를 가진다. </br>
   > 즉, let은 선언된 블럭 ( `{ }` ) 내에서만 유효하고 var은 선언된 함수 전체에서 유효하다.

2. 호이스팅 방식이 다르다.

   > var로 선언된 변수는 `선언과 초기화`가 함께 호이스팅 되어, 선언전 변수 사용시 undefined를 반환한다. let은 `선언`만 호이스팅 되어 선언전 변수 사용시 참조에러가 발생한다.

3. 재선언

   > var는 같은 스코프 내에서 재선언 할 수 있으며, let은 할 수 없다.

4. 전역객체와의 관계
   > 전역스코프에서 var로 선언된 변수는 전역객체(window, global)의 속성이 된다.

⚙️ 추가
- Scope는 무엇인가? <br>
: 스코프는 시별자의 유효범위를 의미한다. 식별자는 자신이 어디에서 선언됐는지에 의해 다른 코드가 자신을 참조할 수 있는 범위를 갖을 수 있다.
