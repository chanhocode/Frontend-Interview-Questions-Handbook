# Javascript 는 어떤 언어 인가?

: 자바스크립트는 `웹페이지를 동적으로` 만들기 위해 사용되는 프로그램 언어이다. 또한 자바스크립트는 브라우저만을 제어하는 언어였지만, Node.js와 구글 V8엔진의 등장으로 자체적으로 어플리케이션을 만들 수 있게 된 언어이다. 자바스크립트는 명령,함수형 객체지향 프로그래밍이 모두 가능한 `멀티 패러다임` 언어이다.

## 주요 특징

- 인터프리터 언어이다.
  > 컴파일 과정 없이 브라우저에서 직접 실행한다.
- 동적타입을 가진다.
  > 변수의 타입이 실행시간에 결정되며, 동적으로 변경될 수 있다.
- 싱글 스레드 논블록킹 언어이다.
  > 비동기 처리를 통해서 싱글 스레드 이지만 블록킹되지 않도록하여 하나의 요청이 완료될 때 까지 기다리지 않고 동시에 다른 작업을 수행하는 것처럼 할 수 있다.

⚙️ 추가

- 자바스크립트가 동적 언어인 이유는?<br>
  : 자바스크립트는 런타임 시 변수의 타입이 결정된다. 즉, 소스가 빌드될 때 자료형이 결정되는 것이 아닌 실행하고 결정된다.

- Blocking & Non-Blocking 이란?
  - Blocking <br>
    블록킹은 직접 제어할 수 없는 작업이 끝날때까지 기다려햐하는 경우 이다. 만약 호출된 함수에서 I/O작업을 요청했을 경우 해당 I/O작업이 완료되기 전까지 아무일도 할 수 없다.
  - Non-Blocking <br>
    직접 제어할 수 없는 작업이 완료되기 전에 제어권을 넘겨주는 경우를 말한다. 만약, 호출된 함수에서 I/O작업을 요청했을 경우 I/O 작업의 처리 여부와 관계없이 바로 다음 작업을 수행할 수 있다.
